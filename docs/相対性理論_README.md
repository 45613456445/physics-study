### clean_srt.py

- **処理内容**: SRTファイルからタイムスタンプ・番号を除去し、重複テキストを削除して連続テキストに変換
- **使用方法**: `python3 clean_srt.py <input.srt> <output.txt>`
- **デフォルト**: 引数なしで実行すると `/tmp/yobinori_subs/relativity.ja.srt` → `文字起こし_生データ.txt`
