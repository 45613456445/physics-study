## 1. 相対性理論とは何か

**相対性理論 = 時間と空間についての革新的な理論**

相対性理論には大きく分けて**2種類**ある：

| | 特殊相対性理論 | 一般相対性理論 |
|---|---|---|
| **発表年** | 1905年 | 1915年 |
| **扱う範囲** | 等速直線運動の世界 | 重力も含めた世界 |
| **難しさ** | 中学数学で理解できる | 大学の物理学科でも学ばないことがある |

一般相対性理論は特殊相対性理論を含んでいる（特殊は一般の一部）。
今回の授業では**特殊相対性理論**のみを扱う。

### アインシュタインについて

- 舌を出した有名な写真は**72歳**のときのもの
- 特殊相対性理論を作ったのは**26歳**のとき
- 12歳の頃に親戚から幾何学の本をもらい、微分積分を独学でマスター

### 相対性理論を学ぶメリット

1. **直感に反することを理屈で受け入れる訓練になる**
   - 物理を勉強すると、直感では分からないが実験結果として正しいことがたくさんある
   - それを論理で受け入れる力がつく

2. **物事を相対的に捉えられるようになる**
   - 「絶対的にこうだ」ではなく「自分から見てこう、相手から見てこう」と考える力

## 4. 2つの原理

特殊相対性理論は、以下の**2つの原理**を出発点にする。

### 原理① 光速度不変の原理

> **どの慣性系（止まっている人or等速直線運動している人）から見ても、光の速さは変わらない**

- 光の速さ = 約30万km/秒（1秒で地球7周半）
- 普通のボールなら、追いかけると遅く見えるし、向かってくると速く見える
- しかし**光だけは違う**：どんなに速く走りながら見ても、光の速さは30万km/秒のまま
- 極端に言えば、光の速さで走りながら光を見ても、光は30万km/秒で進む
- これは何度も実験で確認されている事実（マイケルソン・モーリーの実験など）

**補足：光が特殊なのではなく、「光の速度」が特殊**
- 光の速さ ＝ この世の中の速度の上限
- たまたま光がその上限の速度で進んでいるだけ

### 原理② 特殊相対性原理

> **どの慣性系から見ても物理法則は同じ**

- 電車の中でボールを真上に投げると、真下に落ちてくる（地上と同じ）
- 等速直線運動している電車の中では、止まっているのと区別がつかない
- 「誰が動いているか」は相対的でしかない
  - 電車が動いている？ 自分が動いている？ 地球が動いている？ → 決められない
- **光の法則も含めて**、どの慣性系から見ても物理法則は同じ

### 慣性系とは

**慣性系 = 止まっている人、または等速直線運動をしている人**

- 加速している人はダメ（慣性系ではない）
- 等速で曲がる運動もダメ（加速度を持つ）

## 5. 同時の相対性

**「同時に起きた」ことが、見る人によって同時ではなくなる**

### 思考実験：電車の中の光

1. 非常に速く走る電車の真ん中に**光源**がある
2. 電車の前端と後端に**検出器**がある
3. 光源から光が前後に同時に発射される

**B（電車に乗っている人）から見ると：**
- 光は前後に同じ距離を進むので、**同時に届く**

**A（外から見ている人）から見ると：**
- 電車は前に進んでいる
- 後端の検出器は光に向かってくる → **先に届く**
- 前端の検出器は光から逃げる → **後に届く**
- 光速度不変の原理により、光の速さは前後とも同じ30万km/秒
- したがって、**同時ではない**

### 重要なポイント

- Bにとって「同時」なことが、Aにとっては「同時ではない」
- **同時は絶対的ではなく、相対的**
- これは「時間と空間をバラバラに扱ってはいけない」ことを意味する
- → **時空**（時間と空間を一つにまとめた概念）として扱う必要がある

## 6. 時空図

時間と空間を1つのグラフで表す方法。

- **縦軸**：時間
- **横軸**：位置（空間）
- 止まっているものは縦軸に沿って上に進む（位置は変わらず時間だけ進む）
- 動いているものは斜めに進む

### メモリの取り方のルール

**光が45度になるようにメモリを取る**

- 時間軸で1秒 → 位置軸で30万km（光が1秒で進む距離）
- こうすると光は常に45度の線で表される

### 時空図で分かること

- Aにとっての同時の線と、Bにとっての同時の線は**傾きが違う**
- 動いている人の時間軸と空間軸は、止まっている人の軸に対して**斜めになる**
- → 同時の相対性が視覚的に理解できる

## 7. 時間の遅れ

### 思考実験：光時計

1. 電車の中に「光時計」がある
   - 下から上に光を飛ばして、上の検出器に届いたら「1秒」とする
2. 電車は速度vで等速直線運動している

**B（電車の中の人）から見ると：**
- 光はまっすぐ上に進む
- 距離 = c × t_B（cは光速、t_Bはかかった時間）

**A（外から見ている人）から見ると：**
- 光は**斜めに**進む（電車が動いているから）
- 光の速さはどちらから見ても同じc（光速度不変の原理）
- 斜めの距離 = c × t_A
- 横方向の距離 = v × t_A
- 縦方向の距離 = c × t_B（Bにとっての距離と同じ）

### 三平方の定理を使う

```
(c × t_A)² = (v × t_A)² + (c × t_B)²
```

これを整理すると：

```
t_A² (c² - v²) = c² × t_B²
t_B² = t_A² × (1 - v²/c²)
t_B = t_A × √(1 - v²/c²)
```

### ローレンツ因子

```
√(1 - v²/c²)
```

- vがcより必ず小さいので、この値は**必ず1より小さい**
- つまり **t_B < t_A**（動いているBの時間の方が短い＝遅れている）

### 具体例：v = 0.8c（光速の80%）の場合

```
√(1 - 0.64) = √0.36 = 0.6
```

→ Aにとって**1時間**経ったとき、Bの時計は**36分**しか経っていない！

### 日常で感じない理由

- 日常の速度（時速数百km程度）は光速（30万km/秒）に比べて圧倒的に小さい
- v/c がほぼ0なので、ローレンツ因子がほぼ1になる
- ウサイン・ボルトでもこの効果は0.00...何秒レベル

## 8. 空間の縮み（ローレンツ収縮）

### 思考実験：棒の長さ

1. 静止した棒がある（Aから見た長さ = L_A）
2. Bは速度vの電車に乗って棒を通過する
3. Bから見ると棒が速度vで近づいてくるように見える

### 速さの式を使う

**Aにとって：** v = L_A / t_A
**Bにとって：** v = L_B / t_B

速度vは同じ（相対性原理）なので：

```
L_A / t_A = L_B / t_B
```

時間の遅れの式（t_B = t_A × √(1 - v²/c²)）を代入すると：

```
L_B = L_A × √(1 - v²/c²)
```

### 結論

**動いているものの長さは縮んで見える**

### 具体例：v = 0.6c（光速の60%）の場合

```
√(1 - 0.36) = √0.64 = 0.8
```

→ 静止状態で**1km**の棒が、光速の60%で動いていると**800m**に見える

### 重要な注意

- 物理的に力で押しつぶされているわけではない
- 見え方（測定結果）として縮んでいる
- 電車を降りれば元の長さに戻る

## 9. ミューオンの例 - 実際に起きている証拠

### ミューオンとは

- 素粒子の一種
- 寿命が**約2マイクロ秒**（100万分の2秒）
- 宇宙線が大気にぶつかって上空で生まれる
- ほぼ光速（光速の99.97%）で地上に降ってくる

### 相対性理論なしで計算すると

```
距離 = 速さ × 時間 ≈ 3×10⁸ m/s × 2×10⁻⁶ s ≈ 600m
```

→ ミューオンは**600mしか進めない**はず
→ 上空数十kmで生まれるので、**地上には届かないはず**

### しかし実際には地上で観測されている！

### 2つの視点からの説明

**地上の人（静止）から見ると：時間の遅れ**
- ミューオンはほぼ光速で動いている
- ミューオンの時間はゆっくり進む（寿命が延びて見える）
- だから600mよりずっと遠くまで進める → 地上に届く

**ミューオンの立場から見ると：空間の縮み**
- 自分は止まっていて、地球が近づいてくる
- 地球までの距離がローレンツ収縮で**ぎゅっと縮んでいる**
- だから短い寿命でも地上に届く

→ **どちらの視点で説明しても同じ結果**（相対性原理の美しさ）

## 10. 質量とエネルギーの等価性（E=mc²）

### 核分裂の不思議

- ウラン235（つぶつぶが235個）に何かをぶつけると**核分裂**する
- 分裂後の粒子のつぶつぶの合計数は変わらない（235個のまま）
- しかし**質量を測ると、分裂後の方が軽い**！

### 質量はどこに行った？

- 分裂のときに**莫大な熱エネルギー**が放出される
- 減った質量がエネルギーに変わった

### E = mc²

```
E = mc²
```

- E：エネルギー
- m：質量
- c：光速（約3×10⁸ m/s）

**c²が非常に大きい**ので、ほんの少しの質量がとてつもないエネルギーになる。

### 具体例：1gの物質

```
E = 1×10⁻³ kg × (3×10⁸ m/s)²
  = 9×10¹³ J
  = 90兆ジュール
```

### 原爆との関連

- 広島の原爆で使われたのはウランの核分裂
- 核分裂で減った質量はわずか**0.7g**
- たった0.7gがあれだけのエネルギーを生んだ

### 質量の本質

- **質量 = 加速のしにくさ**
- 重い = 動かしにくい、止めにくい
- エネルギーを与えると質量が増える → 加速しにくくなる
- 光速に近づくと質量（加速のしにくさ）が無限大に近づく → **光速を超えられない**

### 光は質量ゼロ

- 光の質量はゼロ
- 質量ゼロだからこそ、最初から光速で進める
- 質量ゼロのものは常に光速でしか存在できない

### 日常で感じない理由

- ものを温めると本当は重くなっている（エネルギーが質量に変換）
- しかしc²が非常に大きいので、日常レベルの温度変化による質量変化は検出できないほど微小

## 11. 質疑応答・まとめ

### Q: ブラックホールの撮影は相対性理論の証明？
- ブラックホールは重力の話 → **一般相対性理論**の範囲
- 今回やった特殊相対性理論のさらに先の話

### Q: 身近で相対性理論が使われているもの？
- **GPS** が有名な例
- GPS衛星は高速で動いている → 時間の遅れを補正する必要がある
- さらに重力の影響もあるので一般相対性理論の効果も考慮が必要
- 補正しないと位置がずれてしまう

### Q: 光速を超えるものはある？
- 仮想的な思考実験として**タキオン**（質量が虚数の粒子）が考えられている
- ただし因果関係が崩れるなど、多くの問題が生じる
- 現時点では観測されていない

### 今日のまとめ

特殊相対性理論では、**たった2つの原理**（光速度不変の原理＋特殊相対性原理）と**中学数学**（速さの式＋三平方の定理）だけで、以下の驚くべき結論が導ける：

1. **動くものの時間は遅れる** → t_B = t_A × √(1 - v²/c²)
2. **動くものの長さは縮む** → L_B = L_A × √(1 - v²/c²)
3. **エネルギーと質量は等価** → E = mc²

これらはすべて実験で何度も確認されている**事実**であり、ミューオンの観測やGPS衛星の補正など、実際に利用されている。
